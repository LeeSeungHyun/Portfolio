Mealcheck 프로그램

2016.02 ~ 2016.05

회사에서 중식 이용시 외부업체를 이용하였습니다. 프로그램을 만들기 전에는 식단표에다 식사신청을 직접해야 했지만, 개발 후 휴대폰, PC를 이용해서 간단하게 식사신청을 할 수 있게 되었습니다. 기능은 식사 신청, 월별 식사 신청 수 확인, 프로필 수정, 주간, 월간 리포트 등등 있고, 저와 베트남 개발자 한 분과 같이 제작하였습니다.

제가 메인으로 개발을 진행했었고, 어려운 부분이 생겼을 시에는 베트남 개발자께서 서포트를 해주셨습니다. 이 때 당시에는 주간, 월간 리포트 구현에 어려움이 생겨 많이 도와주셨던 기억이 납니다.^^ 웹 개발에 대한 전체적인 그림과 Asp.Net MVC 프레임워크에 대해 잘 알 수 있었습니다. 지금도 계속 서비스를 하고 있습니다. URL : meal.bnftech.com

  • 개발 환경: Window7
  • 사용 기술: C#, Asp.Net MVC5, Bootstrap, Html, Css, Javascript

모바일 발전운전정보 시스템

2016.10 ~ 2017.05

(주)한국남동발전과 진행했던 프로젝트입니다. 수집한 발전소의 데이터를 시각적으로 모바일 화면을 통해서도 볼 수 있게 되었습니다. 저는 프로그램 사용 유저, 기능 사용 통계, Tag정보 관리, 공지사항 등록을 관리하는 Web admin 프로그램을 개발하였습니다. (주)한국남동발전의 요구사항에 따라 Java로 개발을 하게 되었고, 스프링 프레임워크를 이용했습니다.

회사에서는 처음 했던 공공기관 모바일 프로젝트 였고, 기반이 없었기 때문에 사업에 참여했던 사람들 모두 어려워했었습니다. 제가 일하면서 느꼈던 것은 커뮤니케이션이 중요하다고 느꼈습니다. 프로그램을 수정했던 부분이 나도 모르게 남의 코드에 영향을 주었고, 그로 인해 일의 진행이 더뎌지고, 서로 감정 상하는 경우도 있었기 때문입니다. 프로젝트를 하면서 정보를 공유하도록 노력하고, 변경사항이 있으면 미리 알리고자 한다면 다음 프로젝트에서는 더 욱 더 잘할 수 있지 않을까 생각했습니다.

  • 개발환경 : Window7
  • 사용기술 : Java, Spring Framework, Html, Css, Angular.js

자유게시판

  • 사용기술: C#, Asp.Net Core Mvc, Entity Framework, Html, Css, Javascript

무한 스크롤 매모패드

2018.02 ~ 2018.06

angular 프레임워크에 익숙해고자 만들었던 무한 스크롤 매모패드 입니다. 기본적인 로그인, 회원가입, 글쓰기, 수정, 삭제, 좋아요 선택/취소 기능이 있고, 스크롤을 브라우저 끝까지 내리면 글 내용을 추기해서 보여주는 방식으로 페이징 처리를 하였습니다. 또 서버와 연결을 유지하는 방법으로 token을 이용하였습니다.

  • 개발환경: Window10
  • 사용기술: javascript(es6), typescript, angular, node.js(server)

발전소 데이터 감시 프로그램

2018.02 ~ 2018.06

Web기반으로 발전소 데이터 감시 프로그램을 제작하였습니다. 태그(발전소 데이터 신호) 정보를 관리하는 페이지, 공정화면 모니터링, 차트 관리 페이지 기능 구현을 하였고, 전체적인 웹 페이지 레이아웃 제작도 같이 하였습니다.

  • 개발환경: Window10
  • 사용기술: javascript(es6), typescript, angular, Asp.net API
